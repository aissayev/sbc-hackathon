# Rubric crosswalk

A navigation aid. Each rubric line maps to the file or tool that implements it, so a reviewer reading the brief can jump straight to the code. This isn't a status tracker — that lives in [FEATURES.md](./FEATURES.md).

---

> "Simulated customer scenarios across WhatsApp, Instagram, website."

|---|---|

## Penalty avoidance

> "Hardcoded test answers = −10 pts + public note."

Every product, price, lead time, capacity number, and policy comes from a tool call at request time. The grep gate is `bun run audit:hardcodes` — runs on 50+ source files and is wired into the preflight script.
